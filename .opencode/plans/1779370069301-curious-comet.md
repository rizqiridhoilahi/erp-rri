- **Command** - Searchable lists/command palette

- **Chart** - Data visualization with Recharts

#### 2. Color Token Naming Convention
shadcn uses semantic naming like `--primary-foreground` not just `--primary`. Should add:
```
--primary-foreground: ... (for text on primary bg)
--secondary-foreground: ...
--muted-foreground: ...
--destructive-foreground: ...
```

#### 3. Form Patterns - Minor Improvement
Current number input pattern works but could use `value={field.value ?? ""}` instead of `value={field.value != null ? String(field.value) : ''}` (more idiomatic)

#### 4. Toast - Missing Toaster Component
Need to add: `<Toaster />` must be in root layout (app/layout.tsx), not individual pages

#### 5. Typography - Should Use CSS Variables
Current: `text-2xl font-heading font-bold`
Recommended: Use CSS font variables set in globals.css and reference them, not custom classes

#### 6. Performance - Not Documented
- Import components individually (not from index)
- Lazy load heavy dialogs
- DataTable with TanStack Table for complex tables

#### 7. Missing "Blocks" Concept
shadcn/blocks for scaffolding (dashboard-01, login-01, etc.)
